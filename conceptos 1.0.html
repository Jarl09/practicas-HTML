HTML

HTML son las siglas Hyper Text Markup Language, que puede definirse como lenguaje de marcas o lenguaje de hypertexto.

El lenguaje HTML se emplea para crear paginas web. Es muy facil ver el código HTML de una web, la opción exacta cambia de un navegador a otro y también puede cambiar de una versión a otra de un mismo navegador, pero suelen tener un nombre similar.

HTML se descompone de etiquetas que se escriben entre los simbolos menor que y mayor que.

Historia de HTML

Los inicios del lenguaje HTML se remonta al año 1990, cuando Tim Barnes-Lee creó la primera página web.

Versiones de HTML

Tim Barnes-lee definió la primera version de HTML en el año 1991.

En la actualidad, la ultima versión de HTML es HTML 5
